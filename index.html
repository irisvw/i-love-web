Ik heb vandaag een breakdown schets gemaakt (nadat ik mijn visitekaartje helemaal had opgemaakt...). Ik heb verder gewerkt aan mijn visitekaartje - ik heb een beetje responsiveness toegevoegd, ik heb mijn code opgeruimd, en heb alle placeholders vervangen door content. Onderweg naar huis heb ik nog een extra breakpoint toegevoegd voor kleine apparaten zoals smartphones, en heb ik animaties toegevoegd aan de linkjes. Ook heb ik even snel deze site aangemaakt zodat ik een plekje had om dingen op te schrijven.
